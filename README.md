# IP-Tracker
A cyberpunk-inspired IP geolocation tool with a real-time spinning globe, terminal-style logs, and detailed network intelligence.
Features:

i) Real-time 3D globe visualization (D3.js powered)
ii) IP geolocation lookup (country, city, ISP, coordinates)
iii)VPN / Proxy / Hosting detection
iv)eDEX-UI inspired interface
v)Live trace logs & system feedback
vi)Auto-focus globe rotation to target location
vii)Supports IPv4 and IPv6
viii)Local server — no external UI hosting needed

Opens automatically in your browser: Runs on: http://127.0.0.1:7331
